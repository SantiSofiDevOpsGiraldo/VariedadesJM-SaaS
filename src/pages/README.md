# Módulo de Páginas (Pages)

## Propósito
Contiene los componentes de vista principal que corresponden a las rutas de la aplicación.

## Responsabilidades
- Orquestar componentes menores para formar una pantalla completa.
- Gestionar el estado local de la vista.
- Interactuar con los stores globales y hooks de datos (React Query).

## Estructura
Cada archivo `.tsx` representa una página principal accesible vía `react-router-dom`.
